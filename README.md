# necrophonic

Github pages site for my user account - https://necrophonic.github.io

## Building

The site is built with [hugo](gohugo.io)

```shell
$> brew install hugo
$> git clone <repo>
$> cd <repo>
$> hugo
```

This will build a new `public` folder. Commit and push back to `master`
